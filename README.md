# Vulkan C Learning

This project is a simple Vulkan application written in C, created as a learning exercise.

## Project Structure

- `FirstLearn/`: Main project directory containing the Visual Studio project files.
  - `main.c`: Main source file.
  - `triangle.vert` and `triangle.frag`: Shader files.
  - `*.spv`: Compiled shader files.

## Building

1. Open `FirstLearn.sln` in Visual Studio.
2. Build the solution in Debug or Release mode.

## Requirements

- Vulkan SDK
- Visual Studio with C++ support
